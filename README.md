><p>GETITEMINFO(slot,<[&idvar],[#stacksize],[#datavar],[&display]>)</p>

>GETGUINAME(&name)
gets the display name of the current GUI.

>INVFULL(&boolean-string)
returns "empty" or "full" in the string defined.
